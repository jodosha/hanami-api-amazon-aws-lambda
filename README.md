# Hanami::API on Amazon AWS Lambda

## Prerequisites

  * Ruby 2.7+
  * Amazon account
  * Amazon AWS CLI (`aws`)

## Setup

  * Create an [Amazon AWS Lambda function](https://docs.aws.amazon.com/lambda/latest/dg/lambda-ruby.html)
  * Clone this repository
  * Run `bin/setup`
  * Run `bin/deploy`
  * Run `bin/test`
